# Sangeet+ 

Music-App using React Js & Hooks


## Tech Stack

**Client:** React, Hooks, CSS



  
## Run Locally


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Deployment

To deploy this project run

```bash
  npm run deploy
```
